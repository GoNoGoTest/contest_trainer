# HF Contest Trainer

Fork of [oe5sos/contest_trainer](https://github.com/oe5sos/contest_trainer) used for testing improvements and preparing pull requests upstream.

**A free, browser-based SSB & CW contest simulator for amateur radio operators.**

Try original live: **https://oe5sos.github.io/contest_trainer/**

## Features
- Realistic SSB & CW contest simulation (pileup, QSB, QRM, waterfall)
- Adaptive difficulty — starts easy, gets harder as you improve
- Multiple contest modes: CQ WW, CQ WPX, ARRL DX, and more
- Band character profiles (20m / 40m night / 15+10m / 80+160m)
- Extra QRM types: splatter, key-clicks, birdies
- CW realism: chirp & drift simulation
- Works entirely in your browser — no installation needed

## Usage
Simply open `index.html` in any modern browser, or use the live link above.
